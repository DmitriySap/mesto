# Проект: Место
---------------------------------------
Небольшая социальная сеть, в которой есть возможность изменять имя и описание профиля. Проект свёрстан по макету из Фигмы.
---------------------------------------
## Используемые технологии:

1. Grid Layout
2. Flexbox
3. Figma 
4. JavaScript 
5. ООП
6. Git
